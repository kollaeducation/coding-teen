### LESSON 05: TOPOGRAPHY

---

### Objectives
1. Peserta mengerti konsep website responsive.
2. Peserta mampu menggunakan class pada Bootstrap untuk mobile, tablet, desktop dan large screen.

---

### Materi

#### 1. Class Responsive
* Extra small devices Phones (<768px)
  ```html
  <div class="col-xs-6">
    Ini untuk Phones
  </div>
  ```
* Small devices Tablets (≥768px)
  ```html
  <div class="col-sm-6">
    Ini untuk Tablets
  </div>
  ```
* Medium devices Desktops (≥992px)
  ```html
  <div class="col-md-6">
    Ini untuk Desktops
  </div>
  ```
* Large devices Desktops (≥1200px)
  ```html
  <div class="col-lg-6">
    Ini untuk Large Screen
  </div>
  ```
* Digabung semua
  ```html
  <div class="col-xs-6 col-sm-6 col-md-6 col-lg-6">
    Ini gabungan semua
  </div>
  ```

---

### Latihan
1. Buat halaman yang responsive ketika dibuka pada Phone, Tablet, Desktop dan Large Screen.
